# Netsuite RF-Smart & WMS Features

## Bin Transfer by Lot Number

- RF-Smart
  1. Scan Lot number
  2. Select quantity.
  3. Select destination bin.
  4. Confirm transfer.
- WMS
  1. Scan item or bin
  2. Scan item or bin
  3. Scan lot number
  4. Scan destination bin
  5. Confirm transfer.

## Bin Transfer by Bin

- RF-Smart
  1. Scan source bin.
  2. Scan item
  3. Select quantity
  4. Select destination bin.
  5. Confirm transfer.
- WMS
  1. Scan source bin.
  2. Scan item
  3. Select quantity
  4. Select destination bin.
  5. Confirm transfer.
